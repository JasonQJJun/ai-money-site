# AI服务工作室网站

这是一个简单的服务展示和下单网站，用于在闲鱼等平台展示AI服务并收集客户订单。

## 📁 文件结构

```
website/
├── index.html          # 首页（服务展示）
├── order.html          # 下单页面
├── success.html        # 订单提交成功页面
├── styles.css          # 样式文件
├── orders.json         # 订单数据存储（演示用）
└── README.md           # 本文件
```

## 🚀 功能特性

### 1. 首页（index.html）
- 服务项目展示（6种服务类型）
- 关于我们介绍
- 响应式设计，支持移动端
- 渐变色彩设计，现代化UI

### 2. 下单页面（order.html）
- 服务类型选择
- 客户信息收集（姓名、电话、邮箱、微信）
- 需求描述（必填，最少10字符）
- 预算范围选择
- 表单验证
- 错误提示

### 3. 成功页面（success.html）
- 订单详情展示
- 订单编号生成
- 友好的成功提示
- 返回首页链接

## 📱 如何使用

### 本地测试

1. 进入网站目录：
   ```bash
   cd /workspace/projects/workspace/website
   ```

2. 启动本地服务器（Python）：
   ```bash
   python3 -m http.server 8080
   ```

3. 浏览器访问：
   ```
   http://localhost:8080
   ```

### 部署到GitHub Pages

1. 将website文件夹推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择website文件夹作为发布源
4. 访问提供的URL

### 部署到其他平台

- **Netlify**: 直接拖放website文件夹
- **Vercel**: 连接GitHub仓库自动部署
- **云服务器**: 上传文件到web服务器目录

## 🔧 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式设计（渐变、响应式）
- **JavaScript**: 表单处理、页面跳转
- **LocalStorage**: 订单数据存储（演示用）

## 📝 数据存储说明

当前版本使用LocalStorage存储订单数据，仅供演示使用。实际生产环境需要：

1. **后端API**: 使用Node.js、Python或其他后端语言
2. **数据库**: MySQL、MongoDB等存储订单数据
3. **安全性**: 添加CSRF保护、数据验证等
